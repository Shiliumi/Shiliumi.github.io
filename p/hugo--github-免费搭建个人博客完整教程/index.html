Hugo + GitHub 免费搭建个人博客完整教程

从零开始，手把手教你使用 Hugo 静态网站生成器搭建个人博客，包括环境准备、博客创建、主题配置等完整流程

2026-01-20

想要拥有一个属于自己的博客吗？本文将带你从零开始，使用 Hugo 静态网站生成器搭建一个美观、快速的个人博客。

1. 环境准备

在开始搭建博客之前，我们需要准备两个必要的工具：Git 和 Hugo。

1.1 安装 Git

Git 是一个版本控制工具，用于管理我们的博客源码。

下载步骤：

  1. 访问 Git 官网
  2. 下载适合你操作系统的安装程序
  3. 运行安装程序，保持默认设置，一路点击"下一步"即可

安装完成后，打开命令行输入 git --version，如果显示版本号则说明安装成功。

1.2 下载 Hugo

Hugo 是一个用 Go 语言编写的静态网站生成器，以速度快、配置简单著称。

下载步骤：

  1. 访问 Hugo Releases 页面
  2. 选择最新版本的 hugo_extended 版本（Extended 版本支持 SCSS/SASS）
  3. 根据你的操作系统选择对应的压缩包下载
  4. 下载完成后解压到一个你喜欢的目录

提示：推荐下载 Extended 版本，因为很多主题需要 Extended 版本的特性支持。

2. 搭建博客

2.1 创建博客项目

步骤 1：打开命令行

在 hugo.exe 所在文件夹的地址栏输入 cmd，然后按 Enter 键打开命令行窗口。

步骤 2：创建新站点

在命令行中输入以下命令（将 Blog 替换为你想要的博客名称）：

1

hugo new site Blog

这会创建一个名为 Blog 的文件夹，包含 Hugo 博客的基本结构。

步骤 3：进入项目目录

1

cd Blog

将 hugo.exe 复制到刚生成的 Blog 文件夹中，方便后续使用。

步骤 4：启动开发服务器

1

hugo server -D

打开浏览器访问 http://localhost:1313，你会看到一个空白页面，这是因为我们还没有安装主题。

按 Ctrl + C 可以停止服务器。

2.2 配置主题

Hugo 本身不包含默认主题，我们需要选择并安装一个主题。

步骤 1：选择主题

访问 Hugo Themes 浏览并选择你喜欢的主题。

本教程以 Stack 主题为例进行演示。

步骤 2：下载主题

  1. 进入主题的 GitHub 页面
  2. 点击 Code → Download ZIP 下载主题
  3. 解压下载的文件
  4. 将解压后的文件夹重命名为 hugo-theme-stack
  5. 将文件夹移动到 Hugo 根目录下的 themes 文件夹中

步骤 3：复制示例配置

大多数主题都会提供示例配置和内容：

  1. 打开主题文件夹中的 exampleSite 目录
  2. 将 content 文件夹复制到 Hugo 根目录（覆盖原有的）
  3. 将 hugo.yaml（或 config.toml）复制到 Hugo 根目录（覆盖原有的）
  4. 删除根目录下的 hugo.toml（如果存在）
  5. 删除 content/post/rich-content 文件夹（可选）

步骤 4：修改配置文件

打开 hugo.yaml 文件，确保 theme 字段与主题文件夹名称一致：

1

theme: hugo-theme-stack

同时修改其他基本信息：

1
2
3

baseurl: https://example.com/  # 你的网站地址
title: 我的博客                 # 网站标题
languageCode: zh-cn             # 语言代码

步骤 5：预览博客

再次启动 Hugo 服务器：

1

hugo server -D

打开浏览器访问 http://localhost:1313，现在你应该能看到一个漂亮的博客了！

3. 创建第一篇文章

使用以下命令创建新文章：

1

hugo new post/my-first-post/index.md

这会在 content/post/my-first-post/ 目录下创建一个 index.md 文件。

编辑这个文件，添加你的内容：

1
2

+++
title = "我的第一篇文章"
date = "2026-01-20"
description = "这是我的第一篇博客文章"
tags = ["测试"]
categories = ["日记"]
+++

这是文章的正文内容...

保存后，刷新浏览器即可看到新文章。

4. 下一步

现在你已经成功搭建了一个本地博客！接下来你可以：

  1. 自定义主题：修改 hugo.yaml 中的配置项
  2. 添加更多内容：创建更多文章和页面
  3. 部署到线上：使用 GitHub Pages 或其他托管服务

关于如何部署博客，请参考我的另一篇文章：使用 GitHub Actions 自动部署 Hugo 博客

常见问题

Q: 为什么我的主题没有生效？

A: 检查以下几点：

  • hugo.yaml 中的 theme 字段是否与 themes 文件夹中的主题名称一致
  • 主题文件夹是否正确放置在 themes 目录下
  • 尝试删除 resources 文件夹后重新启动

Q: 如何更换主题？

A: 下载新主题到 themes 文件夹，然后修改 hugo.yaml 中的 theme 字段即可。

Q: 文章不显示怎么办？

A: 检查文章的 Front Matter 中是否有 draft: true，如果有，要么删除这一行，要么使用 hugo server -D 命令启动服务器（-D 表示显示草稿）。

总结

通过本教程，你已经学会了：

  • ✅ 安装 Git 和 Hugo
  • ✅ 创建 Hugo 博客项目
  • ✅ 安装和配置主题
  • ✅ 创建和编辑文章

Hugo 的强大之处在于它的速度和灵活性，希望你能享受使用 Hugo 写作的乐趣！
